1.1 Due

Monday Aug 31 before class

1.2 What to do

1.2.1 Practice Linux commands

  • Point your browser to notes from lab1 in your copy of my github repo
  • Review all Linux commands that we covered
  • Spend an hour just practicing them
  • Be sure you're comfortable using them all

1.2.2 Increase your mastery of emacs

It takes going through the tutorial several times before you'll start to gain fluency. Make sure you know how to start emacs, edit and save files, and exit from emacs. Know how to run emacs in the background using &.

1.3 Quiz alert!

There will be a 10 minute quiz on this material at the start of next class. You will be expected to be able to carry out a simple programming tasking involving moving around in the file system, creating directories, editing new python files using emacs, and saving and running them. We will give you the code for the Python program itself. This is really just a quiz to make sure you're able to use the tools you'll need for the rest of the semester.

1.4 Collaboration policy

Help each other out as much as you want to. Ask and answer questions on Piazza. This is an open-everything homework. The point is simply to learn the material. The quiz will test you individually to make sure you're ready to go.
